## ⭐주요 기능
- 자동완성 기능의 검색창을 통한 모델(사용자가 검출하고자 하는 객체를 학습한 모델) 선택 
- 오토 라벨링 된 데이터를 correct할 수 있는 Tool
- 사용자의 correct데이터로 train, autoLabeling 
- 프로젝트 별로 관리되는 워크스페이스 (프로젝트 다운로드, 삭제, 제목 변경, 차트(검출된 객체 정보))
